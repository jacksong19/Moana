# 儿歌创作增强参数设计方案

> 文档日期：2025-12-20
> 设计目标：基于 Suno V5 能力，提供最丰富的创作参数选项，同时通过场景预设和智能联动降低使用门槛

## 一、设计理念

### 核心原则
- **最大自由度**：提供 31 个参数、9 大类别的完整控制能力
- **通俗易懂**：每个参数都有使用说明和智能提示
- **渐进式复杂度**：从预设快捷到专业调参，满足不同用户需求
- **智能辅助**：场景预设一键填充 + 参数智能联动推荐

### 目标用户
- 普通家长：使用场景预设 + 核心参数即可完成创作
- 进阶用户：展开高级设置进行精细调整
- 专业用户：完全自定义所有参数

| 特殊风格 | 游戏音乐、电影配乐、New Age、A Cappella、Gospel |

#### 节奏速度（BPM 滑块）
- 范围：60-180 BPM
- 智能提示随数值变化：
  - 60-80：极慢，适合摇篮曲/冥想
  - 80-100：慢速，适合舒缓放松
  - 100-120：中速，适合日常儿歌
  - 120-140：中快，适合活泼欢快
  - 140-180：快速，适合动感舞蹈

#### 能量强度（滑块）
- 范围：1-10
- 标签：静谧(1-2)、轻柔(3-4)、温和(5-6)、活力(7-8)、激昂(9-10)

### 面板 5：📝 歌词设置

#### 歌词复杂度（滑块 1-10）

| 档位 | 说明 |
|------|------|
| 1-2 极简 | 单词重复，适合 1-2 岁 |

| 档位 | 说明 |
|------|------|

| 5-6 适中 | 适度重复，平衡变化 |
| 7-8 较多 | 副歌多次重复，易记忆 |

### 面板 9：⚙️ Suno 进阶控制

#### 预设组合（快捷选项）

| 预设 | 风格权重 | 创意度 | 适用场景 |
|------|----------|--------|----------|
| 🛡️ 安全稳定 | 80% | 20% | 追求可控结果 |
| ⚖️ 平衡推荐 | 50% | 50% | 通用默认选择 |
| 🚀 大胆探索 | 30% | 80% | 期待惊喜创意 |
| 🎯 精准复刻 | 95% | 10% | 严格按描述生成 |

#### 风格权重（滑块 0-100%）
- 0% = 自由发挥
- 100% = 严格遵循

#### 创意度（滑块 0-100%）
- 0% = 保守稳定
- 100% = 大胆创新

#### 排除风格（多选标签 + 自定义）

常用排除项：
- 不要重金属
- 不要死亡金属
- 不要硬核说唱
- 不要恐怖氛围
- 不要电音失真
- 不要嘶吼人声
- 自定义排除（输入框）

#### 参考曲风描述（文本框）
```
可选：用自然语言描述你想要的风格
示例：像迪士尼动画片里的欢快歌曲，有点爵士风味，适合小朋友跟着跳舞，但不要太吵闹...
```

#### 种子值（可选，高级用户）
- 留空自动生成
- 输入数字种子可复现相似风格

## 八、使用指引设计

### 参数帮助说明

每个参数旁的 💡 图标，点击显示通俗说明：

| 参数 | 通俗说明 |
|------|----------|
| 音乐流派 | 决定歌曲的整体风格，就像选择穿什么类型的衣服 |
| 节奏速度 | 歌曲的快慢，数字越大越欢快动感，越小越安静舒缓 |
| 能量强度 | 歌曲的"热闹程度"，睡前选低能量，玩耍选高能量 |
| 风格权重 | Suno 有多听你的话，100%=完全按你说的，0%=它自由发挥 |
| 创意度 | 允许 Suno 有多少"个人发挥"，高=惊喜多，低=更稳定 |
| 歌曲结构 | 歌曲的"骨架"，决定主歌副歌怎么排列 |
| 重复程度 | 歌词重复多少次，重复多=容易记住，重复少=内容丰富 |
