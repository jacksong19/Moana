# TabBar Icons

This directory contains TabBar icons for the Moana mini-program.

## Current Status

The current files are **placeholder icons** (24x24 gray/coral squares).
Please replace them with proper designed icons before production release.

## Design Guidelines

- Icons should be simple, friendly, and child-appropriate
- Use the warm design style consistent with the app
- Suggested icon styles: home, magic wand/star, bookshelf, user/profile
- PNG format with transparent background
- Recommended: Use 2x or 3x resolution for Retina displays
